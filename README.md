# Q1 Compose State & Recomposition

## Description
This project implements a simple counter screen using Jetpack Compose.  
The screen displays a count starting at 0 and includes two buttons to increment and reset the value.  
State is managed using `mutableStateOf` and preserved across screen rotations using `rememberSaveable`, ensuring the UI updates correctly during recomposition.
